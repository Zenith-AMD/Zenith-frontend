# Zerith Frontend

Frontend for Zerith, an AI-powered contract analysis platform focused on fast risk detection, contract summarization, and practical legal document analysis.

This frontend provides:
- Contract upload interface
- Real-time analysis flow visualization
- Risk highlighting UI
- Responsive modern interface
- Sample contract downloads
- Integration with GPU-backed inference backend

## Tech Stack

- React
- Vite
- TypeScript / JavaScript
- TailwindCSS
- Framer Motion
- Axios

## Features

- Fast document upload workflow
- Animated analysis pipeline
- Responsive design
- Real-time API integration
- Sample contract download support
- Modern glassmorphism-inspired UI
- Optimized for low-latency interaction

Installation
git clone <repo-url>
cd frontend
npm install
Development
npm run dev
Production Build
npm run build
Preview Production Build
npm run preview
Deployment

The frontend is designed for deployment on platforms such as:

Vercel
Netlify

Backend Integration

The frontend communicates with the Zerith backend API for:

Document processing
Contract analysis
Risk detection
AI-generated summaries
Performance Goals

The frontend was intentionally designed around:

Fast perceived responsiveness
Smooth analysis pacing
Minimal interaction friction
Stable large-document handling
Project Philosophy

Zerith focuses on practical AI engineering rather than simply wrapping large language models with prompts. The goal is to build a fast, deployable, usable system with intentional UX and infrastructure decisions.
